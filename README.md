# UERJ / IPRJ / PROINTEC / Chateau 

Projeto de pesquisa desenvolvido no contexto acadêmico da UERJ.

## Objetivo
Explorar e aplicar conceitos de:
- Inteligência Artificial
- LLMs e LangChain
- Arquitetura backend
- Engenharia de software
- Infraestrutura e DevOps

## Estrutura
- `main`: base estável do projeto é um hello wourld que funciona através da chamada [http](http://localhost:8080/hello)
- `study/*`: estudos de caso e experimentos
- `feature/*`: funcionalidades consolidadas

## Tecnologias base
- Java 17
- Quarkus
- Docker
- LangChain

## Observação
Branches de estudo podem conter código experimental.
